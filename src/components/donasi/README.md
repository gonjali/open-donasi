# Halaman Donasi - Pinus Pintar

## Cara Menambahkan Foto

1. Simpan foto-foto program Pinus Pintar dengan nama berikut di folder **public**:
   ```
   /public/pinuspintar-1.jpg
   /public/pinuspintar-2.jpg
   /public/pinuspintar-3.jpg
   /public/pinuspintar-4.jpg
   /public/pinuspintar-5.jpg
   /public/pinuspintar-6.jpg
   ```

2. Format foto yang disarankan:
   - Format: JPG, PNG, atau WebP
   - Ukuran: Minimal 800x600px (rasio 4:3)
   - Ukuran file: Maksimal 1MB per foto untuk performa optimal

3. Jika foto tidak ditemukan, akan muncul placeholder default.
